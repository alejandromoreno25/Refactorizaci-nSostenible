# REFACTORIZACIÓN DE NUESTRA EMPRESA ELEGIDA

## INTRODUCCIÓN A LA EMPRESA

Nuestra empresa elegida es **MONSTRUO DE LABORATORIO**, una empresa la cual vende cookies muy especiales. Esta empresa, tiene una sede en el centro historico de Sevilla, y últimamente ha generado un gran impacto mediático en redes sociales.

## FASE NÚMERO 1: INVENTARIO Y DIMENSIÓN AMBIENTAL.

### 1) Medición Inicial

La empresa de la que vamos a hacer nuestra refactorización, no tiene pagina web creada. Debido a eso, investigando lo que es una pagina web sostenible, nos hemos plantado el objetivo de crear una pagina web que cumpla con los filtros de las Mediciónes propuestas logrando un nivel minimo de A o A+

### 2) Identificación de Bloatware

Aunque la empresa no tenga página web, es importante saber que cosas o elementos hacen que nuestra web sea mucho menos sostenible si no los incorporamos bien y hacemos un uso excesivo como son javascript, videos casi ni visibles de fondo e imagenes sin comprimir.

Al reducir estas cosas, según HTTP Archive reduciriamos nuestro peso medio casi un 80%. Aunque no lo parezca porque creemos que todo esto es invisible, Bootstrap y jQuery requieren que el navegador ejecute miles de líneas de JavaScript antes de mostrar nada, Por lo que para nuestra página web seria una acción muy irresponsable.
